Course Description: This course introduces basic technology (algorithms, architectures, systems) and  research topics in connection with large-scale data management and knowledg discovery techniques for Big Data. The course will start by introducing Big data models, databases and query languages, and cover modern distributed database systems and algorithms and Big data systems adopted in industry and scientific applications. Implementation of a data analysis engine on a standalone machine will be covered and students will learn how to build their own management systems for big data.  Distributed storage and parallel processing and architectures that support data analytics will be examined, and students will learn how to implement a distributed data processing system. The course will also cover critical topics in mining and knowledge discovery of big data, with applications in social analytics, cyber security, and information networks, among others that are already in public eye. 

Learning Objectives/Goals:

  • Understand basic concepts of Big Data techniques, and principles of algorithm designing to cope with Big Data; 
  • Experience real-world problem solving using Big Data computation models;
  • Ability to implement algorithms and systems using open-source Big data softwares.

Prerequisites: Students are expected to have basic programming experiences and knowledge of algorithm design, equivalent to a data structure course CPTS 223 or 233. Some background in basic linear algebra and graph theory is necessary. The course will cover basics in graph theory and relational databases.

Grades: The course consists of lectures (twice a week), 6 assignment and a small project running through the course.  The course project leads to a complete implementation of a small analytic tool over a specific dataset. Students can choose to work in teams for the course project.  A final project report will be given by each team. A reading list will be provided for offline learning.
